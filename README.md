Игра "Memory" (Мемори) - учебный проект

Особенности
- Три темы на выбор: котики, цветочки, машины;
- Отслеживание прогресса: показывает количество открытых пар;
- Счетчик шагов: учитывает каждый ход;
- Таблица рекордов: сохраняет лучшие результаты;

Технологии

- React 19 - компонентный подход;
- Webpack 5 - сборка проекта;
- Babel - транспиляция JSX;
